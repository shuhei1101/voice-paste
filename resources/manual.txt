===================================
  voice-paste v1.0.0
  音声入力＆自動貼り付けツール
===================================

■ 概要
  ChatGPTの音声入力のように、喋った内容を高精度に
  文字起こしするツールです。内部ではGPU対応の高性能AI
  （faster-whisper / large-v3モデル）を使用しており、
  日本語の認識精度が非常に高いのが特徴です。

  さらに、文字起こしした結果を自動的にアクティブなアプリへ
  貼り付け・送信（Enter）まで行うため、テキスト入力欄に
  カーソルを合わせて喋るだけで入力が完了します。

■ 使い方
  1. voice-paste.exe をダブルクリックで起動
  2. タスクトレイにマイクアイコン（青）が表示されたら準備完了
  3. Ctrl+Alt+D で録音開始（アイコンが赤に変化）
  4. もう一度 Ctrl+Alt+D で貼付+送信

■ ホットキー一覧
  Ctrl+Alt+D  ... 録音開始 / 貼付+送信（ペースト後にEnterも送る）
  Ctrl+Alt+V  ... 貼付のみ（ペーストだけ）
  Ctrl+Alt+C  ... コピーのみ（クリップボードにコピーするだけ）
  Ctrl+Alt+Q  ... キャンセル
  Esc         ... キャンセル（ウィンドウにフォーカスがある時）

■ トレイアイコンの色
  青 ... 待機中
  赤 ... 録音中
  緑 ... 文字起こし中

■ 用語集（誤変換の修正）
  yogo.csv を編集することで、Whisperが間違えやすい単語を
  正しく認識させることができます。

  ファイルの場所:
    voice-paste.exe と同じフォルダの yogo.csv
    （初回起動時に自動生成されます）

  .env で別の場所を指定することもできます:
    YOGO_FILE=C:\Users\username\Documents\my_yogo.csv

  書き方（CSVファイル）:
    1行目はヘッダー行です。2行目以降に記載してください。

    ------------------------------------------
    誤変換,正しい表記
    以外に役立つ,意外に役立つ
    検討がつかない,見当がつかない
    ------------------------------------------

    左列: Whisperが間違って出力しがちな表記
    右列: 正しい表記

    ※ 単語単体ではなく文章で書くのがおすすめです。
      AIが文脈を見て判断するため、文章のほうが精度が高くなります。
      例: 「回収」だけだと「ゴミを回収する」まで変わってしまいますが、
          「プログラムを回収する」なら正しい文脈でのみ修正されます。

■ 設定の変更
  タスクトレイアイコンを右クリック → 「設定」で主要な設定を変更可能。
  より詳細な設定は .env ファイルを直接編集してください。
  （初回起動時に自動生成されます）

■ 設定項目の詳細

  [Whisperモデル設定]
    WHISPER_MODEL       ... 使用するモデル（デフォルト: large-v3）
                            tiny/base/small/medium/large-v3 から選択。
                            大きいほど精度が高いが、メモリと処理時間が増加。
    WHISPER_DEVICE      ... 実行デバイス（デフォルト: cuda）
                            cuda=GPU使用、cpu=CPUのみ。
    WHISPER_COMPUTE_TYPE... 量子化タイプ（デフォルト: float16）
                            GPU: float16推奨、CPU: int8推奨。
    WHISPER_BEAM_SIZE   ... ビームサーチ幅（デフォルト: 10）
                            大きいほど精度向上するが速度低下。5〜10が実用的。
    WHISPER_BEST_OF     ... サンプリング候補数（デフォルト: 5）
                            複数候補から最良を選択。精度向上に寄与。
    WHISPER_LANGUAGE    ... 認識言語（デフォルト: ja）

  [VAD設定（音声区間検出）]
    WHISPER_VAD_FILTER          ... 有効/無効（デフォルト: true）
    WHISPER_VAD_THRESHOLD       ... 音声判定の確信度（デフォルト: 0.5）
                                    高くすると雑音を拾いにくくなる。
    WHISPER_VAD_MIN_SPEECH_MS   ... 最小音声長 ms（デフォルト: 250）
                                    これより短い音はノイズとして除外。
    WHISPER_VAD_MIN_SILENCE_MS  ... 最小無音長 ms（デフォルト: 2000）
                                    これ以上の無音で文を区切る（改行位置に影響）。
                                    改行が多すぎる場合は値を大きくする。

  [ホットキー設定]
    RESIDENT_HOTKEY             ... 録音開始（デフォルト: <ctrl>+<alt>+d）
    CONFIRM_HOTKEY              ... 貼付+送信（デフォルト: <ctrl>+<alt>+d）
    CONFIRM_PASTE_ONLY_HOTKEY   ... 貼付のみ（デフォルト: <ctrl>+<alt>+v）
    COPY_ONLY_HOTKEY            ... コピーのみ（デフォルト: <ctrl>+<alt>+c）
    CANCEL_HOTKEY               ... キャンセル（デフォルト: <ctrl>+<alt>+q）

  [動作設定]
    PASTE_ENTER_DELAY   ... 貼付→送信の待機秒数（デフォルト: 0.5）
                            ペースト直後のEnterが効かないアプリ向け。
                            反応が悪い場合は値を大きくする。

  [履歴設定]
    HISTORY_ENABLED         ... 履歴保存の有効/無効（デフォルト: true）
    HISTORY_RETENTION_DAYS  ... 保持日数（デフォルト: 1）
                                この日数を過ぎた履歴は自動削除される。

  [ウィンドウ設定]
    WINDOW_TOPMOST          ... 常に最前面（デフォルト: true）
    WINDOW_POSITION         ... 表示位置（デフォルト: center）
                                center/top-left/top-right/bottom-left/bottom-right
    WINDOW_FOLLOW_CURSOR    ... カーソル側モニターに表示（デフォルト: true）
                                デュアルモニター環境で便利。
    WINDOW_HIDDEN           ... ウィンドウ非表示モード（デフォルト: false）
                                trueにするとトレイアイコンだけで状態表示。

  [プロンプト・用語集]
    PROMPT_FILE ... プロンプトファイル（デフォルト: resources/prompt.txt）
                    自由形式のテキストファイルで、Whisperに文脈のヒントを
                    与えることで認識精度が向上します。

                    使い方の例:
                    - ゲーム実況の文字起こしなら、そのゲームに出てくる
                      キャラクター名・アイテム名・スキル名などを書いておく

                    書き方は自由です。箇条書きでも文章でもOK。
                    基本的にモデルの精度が高いので、何も書かなくても
                    十分使えます。気になる誤認識がある場合だけ追記
                    してください。書きすぎると逆に精度が落ちたり
                    実行時間が延びる可能性があります。

    YOGO_FILE   ... 誤変換修正用の用語集CSV
                    （デフォルト: yogo.csv ※exe直下）
                    相対パス・絶対パスどちらも指定可能。

■ ファイル構成
  voice-paste.exe  ... 実行ファイル
  manual.txt       ... この説明書
  yogo.csv         ... 用語集（初回起動時に自動生成）
  .env             ... 設定ファイル（初回起動時に自動生成）
  _internal\       ... ランタイム（削除不可）
  log\             ... ログ出力先（自動生成）
  history\         ... 録音・文字起こし履歴（自動生成）

■ 動作要件
  - Windows 10/11
  - NVIDIA GPU + CUDAドライバ（GPUモードの場合）
  - CPUのみで使う場合は .env で以下を変更:
      WHISPER_DEVICE=cpu
      WHISPER_COMPUTE_TYPE=int8
