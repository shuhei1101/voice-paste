# voice-paste ガイド

このプロジェクトの構造を理解し、よく行う改修作業にナビゲートする対話スキル。

**重要**: 内部実装の詳細はソースコードを直接読んで確認すること。このガイドはファイルパスと概要のみ記載。

---

## 前提知識

参照ドキュメントはプロジェクト配下の `CLAUDE.md` に配置されている（関連ディレクトリで作業する際に自動読み込みされる）：

- `CLAUDE.md`（プロジェクトルート） — フォルダ構成・アプリケーションフロー・外部ライブラリ
- `voice_paste/CLAUDE.md` — 設定（.env / config.py / settings_gui.py）・GUI・ログ
- `voice_paste/transcription/CLAUDE.md` — faster-whisper・文字起こし処理

---

## 作業フロー

### ステップ1.1（メインメニュー）

条件: スキル起動時。引数がある場合はそれに対応するメニュー項目へ直接遷移する。
入力: なし（引数がある場合はそれを使用）

AI実施タスク:
  - メニューを生成してユーザーに提示する

出力:
```
=== voice-paste ガイド ===

1. プロジェクト概要 — フォルダ構成・アプリケーションフロー
2. 設定一覧 — .env設定項目の確認・編集
3. GUI関連 — 録音モーダル・設定ウィンドウ・トレイ
4. 文字起こし（faster-whisper） — Whisperモデル・文字起こし処理
5. 外部ライブラリ — 使用パッケージ一覧
6. ログ — ログ設定・出力先
```

選択肢:
  1. プロジェクト概要 → ステップ2.1へ
  2. 設定一覧 → ステップ2.2へ
  3. GUI関連 → ステップ2.3へ
  4. 文字起こし → ステップ2.4へ
  5. 外部ライブラリ → ステップ2.5へ
  6. ログ → ステップ2.6へ
  7. その他（入力してください）

---

### ステップ2.1（プロジェクト概要）

条件: メインメニューで「1. プロジェクト概要」が選択された場合
入力: なし

AI実施タスク:
  - プロジェクトルートの `CLAUDE.md` を読み込む（「フォルダ構成」「アプリケーションフロー」セクション）
  - フォルダ構成とアプリケーションフローの概要をユーザーに説明する

出力: フォルダ構成図・アプリケーションフロー・スレッド構成の説明

選択肢:
  1. 特定のファイルの詳細を見る（ファイル名を入力）
  2. メインメニューに戻る → ステップ1.1へ
  3. その他（入力してください）

---

### ステップ2.2（設定一覧）

条件: メインメニューで「2. 設定一覧」が選択された場合
入力: なし

AI実施タスク:
  - `voice_paste/CLAUDE.md` を読み込む（「設定」セクション）
  - 設定項目一覧と編集方法を説明する

出力: 設定項目テーブル・編集手順

選択肢:
  1. 特定の設定を変更したい（設定キーを入力） → ステップ3へ
  2. メインメニューに戻る → ステップ1.1へ
  3. その他（入力してください）

---

### ステップ2.3（GUI関連）

条件: メインメニューで「3. GUI関連」が選択された場合
入力: なし

AI実施タスク:
  - `voice_paste/CLAUDE.md` を読み込む（「GUI」セクション）
  - ファイル一覧・技術スタック・よくある改修を説明する

出力: GUIファイル一覧・技術スタック・改修ガイド

選択肢:
  1. 具体的なGUI改修を依頼する（内容を入力） → ステップ3へ
  2. メインメニューに戻る → ステップ1.1へ
  3. その他（入力してください）

---

### ステップ2.4（文字起こし）

条件: メインメニューで「4. 文字起こし」が選択された場合
入力: なし

AI実施タスク:
  - `voice_paste/transcription/CLAUDE.md` を読み込む
  - ファイル一覧・faster-whisper概要・よくある改修を説明する

出力: 文字起こしファイル一覧・faster-whisper概要・改修ガイド

選択肢:
  1. 具体的な改修を依頼する（内容を入力） → ステップ3へ
  2. メインメニューに戻る → ステップ1.1へ
  3. その他（入力してください）

---

### ステップ2.5（外部ライブラリ）

条件: メインメニューで「5. 外部ライブラリ」が選択された場合
入力: なし

AI実施タスク:
  - プロジェクトルートの `CLAUDE.md` を読み込む（「外部ライブラリ」セクション）
  - ライブラリ一覧と更新・追加方法を説明する

出力: ライブラリテーブル・更新手順

選択肢:
  1. ライブラリの追加・更新を依頼する（内容を入力） → ステップ3へ
  2. メインメニューに戻る → ステップ1.1へ
  3. その他（入力してください）

---

### ステップ2.6（ログ）

条件: メインメニューで「6. ログ」が選択された場合
入力: なし

AI実施タスク:
  - `voice_paste/CLAUDE.md` を読み込む（「ログ」セクション）
  - ログ設定・出力先・よくある改修を説明する

出力: ログファイル一覧・設定・改修ガイド

選択肢:
  1. 具体的な改修を依頼する（内容を入力） → ステップ3へ
  2. メインメニューに戻る → ステップ1.1へ
  3. その他（入力してください）

---

### ステップ3（改修実行）

条件: 各詳細ステップ（2.1〜2.6）でユーザーが具体的な改修を依頼した場合
入力: ユーザーが入力した改修内容

AI実施タスク:
  - 該当ファイルを Read で読み込み、現在の実装を確認する
  - 改修内容を分析し、変更箇所を特定する
  - 必要に応じてファイルを編集する

出力: 改修内容の説明と変更結果

選択肢:
  1. 変更内容でOK → ステップ1.1へ（メインメニューに戻る）
  2. 修正を依頼する（内容を入力）
  3. 変更を取り消す
  4. その他（入力してください）

---

## 動作ルール

1. メニューを表示して選択を待つ
2. 選択された項目の概要を説明する（参照先 CLAUDE.md を Read で読む）
3. ユーザーが具体的な編集を求めたら、該当ファイルを **Read で読んでから** 案内・実装する
4. 作業後はメインメニューに戻るか聞く
5. 実装の内部詳細はこのスキルや CLAUDE.md に書かず、必ずソースを読んで最新の状態を確認する
